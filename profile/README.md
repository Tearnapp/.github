# Tearn

Tearn is a mobile marketplace built to connect students with trusted trainers through a cleaner, more personal training experience.

We are building the product layer for skill-based growth:

- Discover trainers by skill and location
- Connect through a mobile-first experience
- Send and manage training requests
- Pay securely
- Keep communication and progress moving in one place

## What We Build

Tearn combines:

- Trainer discovery
- Student-to-trainer matching
- Secure payments
- In-app chat
- Notifications
- Skill and profile management

## Core Repositories

- `tearn-mobile` — Flutter mobile app for iOS and Android
- `tearn-backend` — Laravel API powering matching, payments, data, and marketplace flows

## Why Tearn

We believe meaningful learning happens when the right student meets the right trainer at the right time.

Tearn is designed to make that connection simpler, faster, and more trusted.
